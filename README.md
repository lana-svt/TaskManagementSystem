# Task Management System

## Описание проекта

RESTful веб-приложение для управления задачами с назначением пользователей, контролем статусов и ограничением нагрузки на сотрудников.

##  Технологии

- **Java 21**
- **Spring Boot 3.5.7**
- **Spring Data JPA**
- **PostgreSQL**
- **Gradle**

##  Функциональность

### Управление задачами
- Создание новых задач
- Просмотр задач по ID
- Обновление существующих задач
- Удаление задач
- Поиск задач с фильтрацией

### Управление статусами задач
- **CREATED** - задача создана
- **IN_PROGRESS** - задача в работе
- **DONE** - задача завершена

##  API Endpoints

### Основные операции с задачами

| Метод | Endpoint | Описание |
|-------|----------|-----------|
| `POST` | `/tasks` | Создать новую задачу |
| `GET` | `/tasks/{id}` | Получить задачу по ID |
| `PUT` | `/tasks/{id}` | Обновить задачу |
| `DELETE` | `/tasks/{id}` | Удалить задачу |

### Управление статусами

| Метод | Endpoint | Описание |
|-------|----------|-----------|
| `POST` | `/tasks/{id}/start` | Перевести задачу в статус IN_PROGRESS |
| `POST` | `/tasks/{id}/complete` | Перевести задачу в статус DONE |

### Поиск и фильтрация

| Метод | Endpoint | Параметры | Описание |
|-------|----------|-----------|-----------|
| `GET` | `/tasks` | `creatorId`, `assignedUserId`, `status`, `priority`, `pageSize`, `pageNum` | Поиск задач с фильтрацией и пагинацией |
